# Money Manager

A personal finance MVP built around dual-pillar spending (Needs vs Wants) with an integrated wishlist buffer and impulse-spending guard.

## Features

- **Onboarding** — Set monthly net income, fixed expenses, and wants budget percentage
- **Dual-Pillar Logger** — Log daily spending classified as Needs or Wants
- **Wishlist Buffer** — Save toward goals using unspent wants budget
- **Cooling-Off Timer** — 72-hour mandatory delay on new wishlist items

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- LocalStorage for persistence (no backend required)

## Build

```bash
npm run build
npm run preview
```
